# Medibox Assets

Ressources partagees pour tous les projets Medibox (CSS, images, composants HTML).

## Structure

```
medibox-assets/
├── css/                    # Feuilles de style partagees
│   └── (a venir)
├── images/
│   └── logo/               # Logos Medibox
│       ├── logo-fond-clair.png
│       └── logo-fond-sombre.png
└── components/             # Blocs HTML reutilisables
    └── INDEX.md            # Liste des composants disponibles
```

## Utilisation

### CSS (GitHub Pages)
```html
<link rel="stylesheet" href="https://elisabethmedibox.github.io/medibox-assets/css/medibox-2025.css">
```

### Images
```html
<img src="https://elisabethmedibox.github.io/medibox-assets/images/logo/logo-fond-clair.png">
```
